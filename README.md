# Projeto 7 — Survival Game

Jogo de sobrevivência desenvolvido em Unity.

## Estado atual do projeto

- Geração de mapa procedural
- Sistema de dia/noite
- Sistema de HP (player, árvores, pedras, mobs)
- IA de mobs noturnos (spawn/despawn automático)
- Inventário, crafting e hotbar
- Sistema de construção
- Corte de árvores e quebra de pedras
- Machado e picareta funcionais
- Respawn do player (perde os itens ao morrer)

## Como puxar o código mais recente (GitHub Desktop)

Se a tua branch foi criada mas não tem nenhum desenvolvimento, faz isto para ficar igual a `main`:

### 1. Abre o GitHub Desktop

### 2. Muda para a branch `main`
- No topo ao centro clica em **Current Branch**
- Seleciona **main**

### 3. Puxa as últimas alterações
- Clica em **Fetch origin** (canto superior direito)
- Depois clica em **Pull origin** se aparecer

### 4. Muda para a tua branch
- Clica em **Current Branch**
- Seleciona a tua branch

### 5. Atualiza a tua branch com o que está na main
- Com a tua branch selecionada, vai ao menu **Branch** (barra de menus em cima)
- Clica em **Update from main** (ou "Merge into current branch" → seleciona `main` ou entao seleciona `Diogo`)

### 6. Abre o projeto no Unity
- No Unity Hub clica em **Open**
- vai até à pasta do projeto e abre

### 7. Depois voltar a trocar para a tua branch

> ⚠️ Se o Unity pedir para reimportar assets, clica em **Yes** — é normal após puxar alterações.
